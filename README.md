# Myntra Clone

A front-end clone of the Myntra homepage, built using **HTML** and **CSS**. This project recreates the layout, navigation bar, banners, and content sections of the Myntra website.

## 🚀 Live Preview

Open `index.html` in your browser to view the project.

## 🛠️ Built With

- **HTML5** – Page structure and content
- **CSS3** – Styling and layout
- **Google Fonts** – Roboto font family

## 📁 Project Structure
├── index.html      # Main HTML file
├── style.css       # Main stylesheet
├── utils.css        # Reusable utility classes (flex, spacing, shadow, etc.)
└── script.js        # JavaScript file

## ✨ Features

- Sticky navigation bar with logo, menu links, and search bar
- Responsive utility classes for flex layouts and spacing
- Banner and category image sections
- Informational content section about the brand

## 📦 Getting Started

To run this project locally:

1. Clone the repository
```bash
   git clone <https://github.com/riya-sharma-09/Myntra-clone.git>
```
2. Navigate into the project folder
```bash
   cd myntra-clone
```
3. Open `index.html` in your browser — no build steps or installation needed.
